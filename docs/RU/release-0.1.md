## Релиз 0.1

Дата: 2026-03-11

### Backend

- **Admin**: доработан `admin.service` и роуты для управления UAT-тестами и административными функциями.
- **Auth**: уточнена логика аутентификации и связанных сервисов.
- **Boards / Issues / Projects**: обновлены сервисы досок, задач и проектов, улучшена работа с данными и бизнес-правилами.
- **Seed-данные**: расширен скрипт `seed` для генерации демонстрационных данных.
- **Redis**: доработана конфигурация и инициализация Redis.
- **Обработка ошибок**: улучшен middleware обработки ошибок.
- **ESLint**: добавлены и обновлены конфигурации ESLint для backend.

### Frontend

- **Навигация и страницы**: актуализированы страницы `Admin`, `Dashboard`, `Board`, `IssueDetail`, `ProjectDetail`, `Projects`, `Sprints`, `Teams`, `Time`, `Login`.
- **UAT**: добавлены API-клиент и страница для UAT-тестов, компонент онбординга UAT.
- **Состояние**: обновлён store задач, добавлен store для UAT-onboarding.
- **Стиль**: расширен и переработан `styles.css` для поддержки нового UI.
- **Тесты и конфигурация**: добавлены e2e-сценарии, Playwright-конфиг, ESLint-конфиг для фронтенда.

### Прочее

- **Скрипты разработки**: добавлен вспомогательный скрипт `tt-dev.sh`.
- **Документация**: обновлён план пересборки в `REBUILD_PLAN_V2.md`.

### Статус спринтов (на 2026-03-11)

- **Sprint 1** — закрыт, фундамент (Auth + Users + Projects + Issues) в боевом состоянии.
- **Sprint 2** — закрыт, функциональность Kanban Board, Sprints, Time tracking и Comments включена в релиз 0.1.
- **Sprint 3** — закрыт, функциональность Teams, Admin, базовые Reports и доработки Redis включены в релиз 0.1.
