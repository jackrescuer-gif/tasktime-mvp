## Sprint 1 — Auth, Projects, Issues

### 1.1 Вход в систему

- [ ] Открыть `/login`
- [ ] Ввести `admin@tasktime.ru` / `password123` → должен открыться Dashboard
- [ ] Ввести неверный пароль → ошибка "Invalid credentials"
- [ ] Нажать "Logout" → редирект на `/login`, токен удалён

**Роли:** все аккаунты

---

### 1.2 Проекты

- [ ] **ADMIN:** создать новый проект (название + уникальный ключ 2–6 заглавных букв)
- [ ] **ADMIN:** изменить название проекта
- [ ] **MANAGER:** создать проект — должно работать
- [ ] **USER:** попытаться создать проект — кнопка «Новый проект» отсутствует или запрос возвращает 403
- [ ] **VIEWER:** открыть список проектов — только просмотр, нет кнопок создания

---

### 1.3 Задачи (Issues)

- [ ] **ADMIN/MANAGER/USER:** создать задачу типа TASK с названием и описанием
- [ ] Создать EPIC → STORY внутри него → TASK внутри STORY (проверить иерархию)
- [ ] Создать BUG
- [ ] **ADMIN:** удалить задачу
- [ ] **MANAGER:** попытаться удалить задачу → 403
- [ ] **USER:** попытаться удалить чужую задачу → 403
- [ ] Изменить приоритет (LOW / MEDIUM / HIGH / CRITICAL)
- [ ] **MANAGER/ADMIN:** назначить задачу на другого пользователя
- [ ] **USER:** попытаться назначить задачу → кнопка недоступна или 403

---

### 2.2 Спринты

- [ ] **ADMIN/MANAGER:** создать спринт в проекте с целью

- [ ] Закрыть спринт — незавершённые задачи должны оставаться в бэклоге
- [ ] **USER:** попытаться создать спринт → 403

## Sprint 4 — AI, Интеграции, Экспорт

### 4.1 AI Estimate

> Требует `ANTHROPIC_API_KEY` в backend `.env`. Без ключа — ответ 503.

- [ ] Открыть любую задачу (TASK, BUG, STORY, EPIC, SUBTASK)
- [ ] В правой панели найти блок **AI Estimate**
- [ ] Нажать кнопку **AI Estimate**
- [ ] Дождаться ответа (3–10 сек)
- [ ] Результат: часы + confidence (low/medium/high) + обоснование
- [ ] Поле "Estimated hours" в задаче обновилось
- [ ] **VIEWER:** блок AI Estimate отсутствует
- [ ] Задача с подробным описанием → confidence = `high`
- [ ] Задача без описания → confidence = `low`

---

### 4.2 AI Decompose

- [ ] Открыть задачу типа **EPIC** → в правой панели кнопка "Decompose into STORYs"
- [ ] Открыть задачу типа **STORY** → кнопка "Decompose into TASKs"
- [ ] Открыть задачу типа **TASK** → блок AI Decompose отсутствует
- [ ] Нажать кнопку декомпозиции → AI предложил 3–7 подзадач
- [ ] Снять галочку с одной подзадачи → выбрано N-1
- [ ] Нажать "Create selected (N)" → задачи созданы
- [ ] Обновить страницу → дочерние задачи появились в иерархии

---

### 4.3 Экспорт отчётов

- [ ] Войти как **ADMIN** → Admin → Reports
- [ ] Выбрать проект

- [ ] Нажать **Issues PDF** → файл скачался, открывается как таблица задач
- [ ] Нажать **Time CSV** → файл скачался с логами времени
- [ ] **USER:** кнопки экспорта отсутствуют
- [ ] Выбрать спринт → экспорт фильтруется по спринту
- [ ] CSV использует кодировку UTF-8 (кириллица корректна)

## RBAC итоговая проверка

### Сценарий: USER не может назначить задачу

1. Войти как `dev@tasktime.ru`
2. Открыть любую задачу
3. Попытаться изменить Assignee → кнопка недоступна или возвращает 403

### Сценарий: VIEWER только читает

1. Войти как `viewer@tasktime.ru`
2. Открыть проект — нет кнопок "New Issue", "New Sprint"
3. Открыть задачу — нет кнопок Save, Start timer, AI Estimate
4. Попытаться POST /api/projects/:id/issues через DevTools → 403

### Сценарий: MANAGER не может удалить проект

1. Войти как `manager@tasktime.ru`
2. DELETE /api/projects/:id → 403
