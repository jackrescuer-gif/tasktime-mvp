# Документация Flow Universe

Документация разнесена по языкам:

- **ENG/** — версия на английском (включая архитектурные документы и текущие руководства).
- **RU/** — та же структура, на русском языке.

## Содержимое

В каждой папке (ENG и RU):

- В корне: API.md, USER_GUIDE.md, ADMIN_GUIDE.md, ROADMAP_TZ.md, PIXEL_AGENTS_AND_TASKS.md, SETUP_VSCODE_PIXEL_AGENTS.md, mvp_jira_*.md.
- **architecture/** — документы по архитектуре MVP (текущая архитектура, доменная модель, предложение по модульному монолиту, план миграции, правила для Cursor, AI-агенты, движок задач, Kanban, производительность, итоговая сводка).

Для навигации по архитектуре удобно начинать с **SYSTEM_ARCHITECTURE.md** в соответствующей языковой папке.
