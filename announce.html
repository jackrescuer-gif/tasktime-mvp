Функциональность
Что есть в системе
📋

Задачи и иерархия

EPIC → STORY → TASK → SUBTASK, BUG. Ключи проекта (TTMP-42), статусы, приоритеты, история изменений.

🗂️

Kanban-доска

Колонки по статусам, drag-and-drop, мгновенное сохранение порядка и статуса задач.

🏃

Спринты (Scrum)

Создание, старт, закрытие спринтов. Перемещение задач из бэклога. Один активный спринт на проект.

⏱️

Учёт времени

Таймер в один клик + ручной ввод. Страница My Time с аналитикой по дням. Трёхуровневая сегментация: HUMAN / HUMAN_AI / AGENT.

🤖

AI-модуль

Оценка трудоёмкости задачи по описанию. Автоматическая декомпозиция EPIC/STORY на подзадачи.

📊

Отчёты и экспорт

Отчёты по задачам и времени. Экспорт в CSV и PDF. API выгрузки открытых задач.

💬

Комментарии

CRUD-комментарии к задачам с проверкой прав. История изменений карточки из audit log.

👥

Команды и Admin

Управление командами и участниками. Панель администратора: пользователи, роли, системные настройки.

🔗

Интеграции

GitLab webhook — автообновление статусов при push/merge. Telegram-бот — уведомления о задачах и статусах.

Роли в системе
🔑

Admin

Полный доступ: управление пользователями, ролями, всеми проектами, системными настройками. Audit log.

📐

Manager

Управление проектами и задачами. Создание спринтов, назначение исполнителей, просмотр всех задач команды.

💻

User

Работа со своими задачами: создание, изменение статусов, трекинг времени, комментарии.

👁️

Viewer

Только чтение — все проекты, задачи и отчёты. Режим для CIO и наблюдателей без права редактирования.
